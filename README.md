### Paradigmas

Repositório criado para guardar, em especial, o trabalho desenvolvido na disciplina de Paradigmas da Programação, Universidade Federal do Paraná, por Mardoqueu F. Nunes.


O trabalho consistem um jogo de tabuleiro: Banco Imobiliario.

##### OBS
- O jogo possui algumas limitações como: Apenas uma rodada, não é possível escolher nome para jogadres e nem cor (está setado uma constante) e está configurado para rodar apenas em monitores Full HD.
